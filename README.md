# Customer Segmentation
*A virtual retail analytics internship experience.*

### Overview
This is a data analysis project aimed at understanding retail sales data, where comprehensive data analysis will be performed on transaction and customer data to derive insights for strategic decision-making. In this early stage of analysis, the primary focus is on performing high-level data checks, deriving additional features, defining relevant metrics, and laying the groundwork for deeper analysis. Ultimately, the goal is to formulate actionable recommendations based on the findings.

### Technical details
Programming language: Python
